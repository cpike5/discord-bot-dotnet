Troubleshooting

I'm not receiving DMs from the bot

If the bot cannot send you a direct message, it's likely due to your Discord privacy settings. Here's how to fix it:

  • Right-click on the server name in Discord
  • Select "Privacy Settings"
  • Enable "Allow direct messages from server members"
  • Try the /register command again

My invite code isn't working

If your invite code is being rejected, it could be due to:

  • Expired Code: Codes expire after 7 days. Request a new one with /register
  • Already Used: Each code can only be used once. If you already registered, try logging in instead
  • Typo: Double-check that you copied the entire code correctly
  • Not Linked to Discord: The code must be used by the same Discord account that requested it

I didn't receive the verification email

If you haven't received your verification email:

  • Check Spam Folder: The email might have been filtered as spam
  • Wait a Few Minutes: Email delivery can sometimes be delayed
  • Check Email Address: Make sure you entered the correct email during registration
  • Request New Email: You can request a new verification email from the login page
