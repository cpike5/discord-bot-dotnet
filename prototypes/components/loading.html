Implementation Notes

  • Spinners: Use for short waits (under 2 seconds) or when duration is unknown
  • Progress bars: Use when you can show actual progress percentage
  • Skeleton loaders: Use for initial page loads to reduce perceived wait time
  • Dots/Pulse loaders: Use for subtle, non-intrusive loading states
  • Loading overlays: Use when blocking user interaction is necessary
  • Always provide context with loading text when possible
  • Match loader size to the context (small for inline, large for full-page)
  • Use skeleton loaders that match your actual content structure
