Feature Cards Component

Features

  • 3-Column Grid: Responsive layout with 3 columns on desktop, 2 on tablet, 1 on mobile
  • Hover Effects: Cards lift and show color transition on hover
  • Icon + Title + Description: Clear information hierarchy
  • Color-Coded Icons: Each card has unique gradient background for icons
  • Learn More Links: Animated arrow links for navigation
  • Top Border Animation: Colored bar appears on hover
  • Two Variants: Default (white background) and Filled (gradient background)
  • Section Header: Badge, title, and description for context

Implementation Notes

  • Uses design tokens from tokens.css
  • Grid columns: 3 (desktop) ‚Üí 2 (tablet ‚â§1024px) ‚Üí 1 (mobile ‚â§768px)
  • Card padding: --space-8 on desktop, --space-6 on mobile
  • Icon size: 64px on desktop, 56px on mobile
  • Replace emojis with icon fonts (Font Awesome, Heroicons, etc.) or SVG icons
  • Hover effects use transform: translateY(-4px) for elevation
  • Top border animation uses transform: scaleX()

Customization Options

  • Grid Columns: Change grid-template-columns for 2, 4, or more columns
  • Card Count: Add or remove cards as needed (currently shows 6 features)
  • Icon Style: Swap emojis for icon library, images, or SVGs
  • Icon Background: Each card can have unique gradient colors
  • Card Style: Choose between default (white) or filled (gradient) variant
  • Link Style: Remove "Learn more" links if not needed
  • Section Badge: Customize badge text and color
