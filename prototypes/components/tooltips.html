Implementation Notes

  • Tooltips appear on hover and focus for accessibility
  • Position variants: top, bottom, left, right
  • Arrow automatically points to the trigger element
  • Dark variant (default) for light backgrounds
  • Light variant with border for dark or colored backgrounds
  • Large variant supports multi-line content
  • Tooltips are non-interactive and dismiss when hover ends
  • Use tooltips for supplementary information, not critical content
  • Keep tooltip text concise and scannable
